# DMCF-recsys
# 项目环境与依赖说明

本项目基于以下开发环境与依赖库构建：

## ✅ 系统环境

- 操作系统：Ubuntu 22.04
- Python 版本：3.12
- CUDA 版本：12.4
- PyTorch 版本：2.5.1（支持 CUDA 12.4）

## 📦 所需 Python 库

请使用以下命令安装项目所需依赖：

```bash
pip install pandas numpy matplotlib scikit-learn
pip install torch==2.5.1+cu124 -f https://download.pytorch.org/whl/torch_stable.html
